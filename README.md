# D&D Character Sheet Manager

A web-based Dungeons & Dragons companion application designed to help players create and manage characters, organize worlds, and track progression throughout their campaigns.

## Features

- User Account Creation & Login
- Character Creation
- Character Management
- World Creation
- Join Existing Worlds
- Character Progression Tracking
- Experience Management
- D&D-Themed User Interface
- Local Storage Data Persistence

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Browser Local Storage

## How to Use

1. Create an account.
2. Log in to the application.
3. Create or join a world.
4. Create a character.
5. Manage character information and progression.
6. Track experience and campaign data.
